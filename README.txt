
# MyGymBro

MyGymBro is a cross-platform gym workout tracking app built using Flutter and FastAPI. This is the starter structure for the full-stack app.

## 💡 Features (MVP)
- Create, track, and save workout presets
- Customize reps, sets, and weight
- Daily workout reminders
- Simple sakura-themed UI (pink light mode, starry night dark mode)

## 🔧 Tech Stack

### Frontend
- Flutter (Dart)
- Supports Android & iOS

### Backend
- FastAPI (Python)
- PostgreSQL or SQLite (for MVP)
- Dockerized for easy deployment

## 🚀 Future Features
- AI-generated custom workout plans
- Progressive overload suggestions
- Food/macros tracking (premium)

## 📁 Structure

- `frontend_flutter/`: Contains Flutter mobile app
- `backend_fastapi/`: Contains FastAPI backend with API routes
